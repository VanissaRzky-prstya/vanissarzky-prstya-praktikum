Pengertian

Semantic HTML merupakan HTML dengan elemen-elemen yang menyatakan makna atau tujuan dari elemen itu sendiri

Semantic HTML ini baru tersedia di HTML 5. Penggunaan Semantic HTML dapat menggantikan fungsi div dengan atributnya, sehingga baris kode jadi lebih ringkas.
